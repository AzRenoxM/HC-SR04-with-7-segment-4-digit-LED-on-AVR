# HC-SR04-with-7-segment-4-digit-LED-on-AVR
HC-SR04 with 7 segment 4 digit LED on Atmega329PB, by using the module HC-SR04, it calculates the distance and display it on a 7 segment x 4 digits LED display, a keyboard with 3 buttons is connected on it to set up with them 3 refresh rate, the last is stop
